# แผนงานโปรเจกต์ AI Saumu Lucky (แอปพลิเคชันสายมู)

## 📌 ที่มาและภาพรวม (Background & Motivation)
อ้างอิงจากความจำถาวร: คุณมีโปรเจกต์ชื่อ **'AI Saumu Lucky'** (พาท: `E:\flutterapp\ai_saumu_lucky`) ที่พัฒนาด้วย **Flutter (Dart)** 
เป้าหมายคือการสร้างแอปพลิเคชันสายมูที่ใช้งานง่าย โดยมีฟีเจอร์หลักดังนี้:
1. **ทํานายฝันเป็นเลขเด็ด** (ใช้ AI จาก Gemini API วิเคราะห์และตีเลข)
2. **ดูดวงรายวัน** (Daily Horoscope)
3. **ระบบโฆษณา** (AdMob Integration เพื่อสร้างรายได้)

ธีมของแอปจะเน้นความสว่าง สบายตา และเข้าถึงง่าย (ขาว-ทอง) เนื่องจากคุณเพิ่งเคยทำแอปครั้งแรก การพัฒนาจะทำไปแบบเป็นขั้นตอน พร้อมกับการอธิบายให้คุณเข้าใจการทำงานของ Flutter ไปด้วย

## 🛠️ ขั้นตอนการดำเนินงาน (Implementation Plan)

### เฟส 0: การเตรียมเครื่องมือ (Installation & Setup) - *สำหรับมือใหม่*
- **ติดตั้ง Flutter SDK:** ใช้คำสั่ง `winget install Flutter.Flutter` (เปรียบเสมือนการซื้ออุปกรณ์ก่อสร้าง)
- **ติดตั้ง Git:** (ติดตั้งเรียบร้อยแล้ว) เพื่อใช้จัดการเวอร์ชันของโค้ด
- **ติดตั้ง VS Code:** (ถ้ายังไม่มี) ใช้ `winget install Microsoft.VisualStudioCode` เพื่อใช้เป็นสมุดจดโค้ด
- **ตั้งค่า PATH:** บอกให้ Windows รู้จักที่อยู่ของ Flutter (ผมจะแนะนำวิธีทำหลังจากโหลดเสร็จ)

### เฟส 1: การจัดการความจำ (Memory Cleanup)
- บันทึกไฟล์แผนงานฉบับนี้เป็นนามสกุล `.md` ลงในโฟลเดอร์โปรเจกต์

### เฟส 2: การตั้งค่าโปรเจกต์ (Project Setup)
- สร้างโปรเจกต์ Flutter ในโฟลเดอร์ `E:\flutterapp\ai_saumu_lucky`
- ติดตั้ง Packages ที่จำเป็น: 
  - `google_generative_ai` (สำหรับเชื่อมต่อ Gemini API)
  - `google_mobile_ads` (สำหรับแสดงโฆษณา AdMob)
- ปรับโครงสร้างไฟล์ `lib/main.dart` ให้เป็นโครงร่างพื้นฐานที่เข้าใจง่ายที่สุด

### เฟส 3: การพัฒนาส่วนแสดงผลและฟีเจอร์หลัก (UI & Core Features)
- **UI Design:** ออกแบบหน้าจอหลัก (Home Screen) โทนสีขาว-ทอง 
- **Dream to Lucky Number:** สร้างหน้าจอสำหรับกรอกข้อความความฝัน และเชื่อมต่อกับ Gemini API เพื่อให้ AI วิเคราะห์คำทำนายและตีเลขเด็ด
- **Daily Horoscope:** สร้างหน้าจอแสดงคำทำนายดวงประจำวันแบบเรียบง่าย

### เฟส 4: ระบบสร้างรายได้ (AdMob Integration)
- ตั้งค่า **Google AdMob** ให้กับโปรเจกต์ (แก้ไข `AndroidManifest.xml` และ `Info.plist`)
- สร้างแบนเนอร์โฆษณาทดสอบ (Test Banner Ad) และนำไปวางที่ด้านล่างของหน้าจอหลัก
